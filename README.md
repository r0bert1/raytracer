## Käyttöohje

Suorita juurikansiossa ensin

```
./gradlew build
```

Aja sitten ohjelma suorittamalla

```
./gradlew run
```

Testit ajetaan komennolla

```
./gradlew test
```

(Käytä windowsilla gradlew.bat ./gradlew sijaan)
